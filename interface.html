Interface¶

The interface api exposes Firewall and DDNS and Workgroup related settings present in the the NAS device.

Firewall Object¶

The following represents a firewall object:

[
 {
    "id": 1,
    "ip": "10.82.0.202",
    "type": "allow"
 },
 {
    "id": 2,
    "ip": "10.82.0.203",
    "type": "deny"
 }
]

firewall Object Details

Name Value
id The primary key unique id by which the network can be identified.
ip IP address
type
value which says whether the current

ip address is allowed or denyed

GET firewall¶

Returns firewal object containing the ip firewal present in the NAS device.

Resource URL http://<nas_box_ip_address>/index.php/firewal/api

Input –> None

Response –> Array of Firewal objects

POST - Add IP¶

Takes Firewal object containing the ip address to add

Resource URL –> <nas_box_ip_address>/index.php/firewal/api

Input –> Firewal object

Response –> Firewal object which got added.

POST - Delete IP¶

Takes Firewal object containing the ip address to delete

Resource URL –> <nas_box_ip_address>/index.php/firewal/api

Input –> Firewal object

Response –> Firewal object which got deleted.

DDNS¶

The interface api exposes Firewall and DDNS and Workgroup related settings present in the the NAS device.

DDNS Object¶

The following represents a DDNS object

{
    "id": 1,
    "dns_server": {
        "www.no-ip.com",
        "www.dyndns.com"
    },
        "dynamic_dns": {
        "username": "",
        "domain_url": "",
        "password": "",
        "enabled": true,
        "server": "www.dyndns.com"
    }
}

DDNS Object Details

Name Value
id The primary key unique id by which the network can be identified.
dns_server This is the default dns server available
dynamic_dns This is user specified dns server
username username for particular server
domain_url says complete domain name
password says password for the user
enabled it says DDNS is enabled or not Boolean
server This object says the current server being used with username

GET DDNS¶

Input –> DDNS object

Response –> DDNS object which got added.
