Printer¶

The printer api exposes printer related all services in the the NAS device.

Printer Objects¶

The following represents a printer objects:

{
    "id": 1,
    "spooler_volume_path": "/VolumeData/volume1",
    "printers": [
        {
            "name": "Canon_MX410_series_USB_1",
            "description": "MFP printer",
            "location": "Local Printer",
            "printer_state": "idle",
            "is_shared": "1",
            "accepting_jobs": "1"
        }
    ]
}

Printer Object Details

Name Value
id
The primary key unique id by which the printer can be
identified
spooler_volume_path This object contains current volume spooler path
printers This object conatins all available printers
name Printer name
description Printer description
printer_state This object conatins “printing” and “idle” state.
is_shared Boolean by default value is “1” it depict printer is shared or not.
accepting_jobs Boolean it depict prinitng jobs. It have either rejected or accepted
