Notifications - Targets¶

The target api exposes interfaces related to targets to display events in the the NAS device.

Targets Object¶

The following represents a targets object:

{
    "email_recipients": [
     {
        "username": "",
        "email_id": "",
        "id": 1
     }
    ],
    "smtp": {
        "authentication": {
            "username": "",
            "enabled": false
        },
        "ssl_tls": {
            "type": "ssl",
            "enabled": true
        },
        "port": 587,
        "server_name": "smtp.gmail.com",
        "sender": ""
    },
    "id": 1,
    "targets": [
     {
        "enabled": true,
        "id": 1,
        "name": "LCD"
     }
    ]
}

GET Targets¶

Returns Targets object containing the target present in the NAS device.

Resource URL http://<nas_box_ip_address>/index.php/targets/api

Input –> None

Response –> Array of Targets objects

POST - Save Target¶

Takes Target object containing the Target to display.

Resource URL –> <nas_box_ip_address>/index.php/target/api

Input –> Targets object

Response –> Targets object
